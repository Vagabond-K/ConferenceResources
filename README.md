# 컨퍼런스, 세미나, 밋업 발표자료

## ⚠️ 재사용에 대하여
모든 발표자료 파일은 Power Point 원본으로 공유하고 있으며, 대부분 저의 동의 없이 마음대로 재사용 가능합니다. 단, 제가 소속되어 있는 회사 및 회사 제품과 관련한 이미지나 동영상은 해당 회사의 소유물이므로 단순 소개를 위한 용도 외 2차 가공은 지양해 주시기 바랍니다.  

참고로 Power Point로 작성한 발표 자료에서 사용한 글꼴 파일을 [Fonts 디렉토리](Fonts/)에 첨부했습니다. 필요하신 분은 다운로드하여 사용하시면 되겠습니다.
